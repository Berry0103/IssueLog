常见问题

要添加新设备，请按照以下步骤操作：
1. 进入"设备管理"模块
2. 点击"添加设备"按钮
3. 填写设备基本信息，包括设备名称、型号、序列号等
4. 配置设备连接参数，如IP地址、端口号等
5. 选择设备所属分组和位置
6. 点击"保存"完成添加
新设备添加后，系统会自动尝试连接并获取数据。

设备显示离线可能有多种原因，请按以下步骤排查：
1. 检查设备是否通电并正常运行
2. 检查网络连接是否正常，设备IP是否可达
3. 检查防火墙设置是否阻止了设备与服务器的通信
4. 重启设备尝试恢复连接
5. 如问题持续，请联系技术支持。

设置设备告警阈值的步骤：
1. 在"设备管理"中找到需要设置的设备，点击"查看"
2. 在设备详情页面中，找到"告警设置"选项卡
3. 对各项参数设置合适的告警阈值，包括警告阈值和错误阈值
4. 选择告警通知方式（邮件、短信等）
5. 点击"保存设置"完成配置
系统会根据设置的阈值监控设备状态，并在超出阈值时发出相应告警。

导出设备数据报表的方法：
1. 进入"数据统计"模块
2. 设置需要导出的数据范围、时间周期和设备范围
3. 点击"生成报表"按钮
4. 报表生成完成后，点击"导出报表"按钮
5. 选择导出格式（Excel、PDF等）
6. 系统会生成并下载相应的报表文件。

添加新用户并分配权限的步骤：
1. 进入"用户管理"模块（需要管理员权限）
2. 点击"添加用户"按钮
3. 填写用户基本信息，包括用户名、姓名、邮箱等
4. 设置初始密码
5. 从下拉列表中选择合适的用户角色
6. 点击"保存"完成添加
系统会根据选择的角色自动分配相应的权限。如需自定义权限，请在"角色与权限管理"中进行配置。
